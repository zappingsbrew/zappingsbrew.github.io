Download
1. Install Rainmeter.
2. Click the Download button below.
3. Run the .rmskin installer.
4. Read through the installer. If you do not deselect the "Load included layout" checkbox, Rainmeter will load the included Mondtholomewtholomew layout and override any other skins you had loaded. This does not remove the skins you had, just unloads them. If you do not have skins loaded or want to load the Mondtholomewtholomew layout, leave the checkbox selected.
5. Click install.
Download
Requirements: Latest Rainmeter version on Windows 7 or above.

FAQs

What?

Mondtholomewtholomew is a "successor" of Reisir's ever popular Mondtholomew, which is also a "remix" of ApexXx-SenSei's ever popular Mond. It is made to continue development for the now-archived Mondtholomew, fix issues, and provide a generally better experience.

Why?

Reisir stopped working on Mondtholomew on March 11, 2024, so I have to continue working on it, fix weather skin, and improve on it.

List of things Mondtholomewtholomew fixes:

Weather
Mondtholomew's api key is now outdated in 2025, it is finally fixed so you don't have to manually replace the API key every time you install Mondtholomew.

Font
Mondtholomewtholomew includes the font but doesn't use it by default. You may enable it by editing the settings, but you can finally toggle back to the new font again.

Centering
Mond isn't centered properly inside the skin window. It shifts position when the day changes. Mondtholomew centers itself properly and provides right click centering options to accurately set the position of the skin to the center of the screen.

Translation
Mondtholomewtholomew uses the local language of the users computer by default for the clock, so it's automatically translated. You may also change the language used for Weather.com queries to get weather information in your language.

Player
Mondtholomewtholomew comes with WebNowPlaying out of the box, making it work with the WebNowPlaying browser extension and Spotify by default.

How do I make Mondtholomewtholomew smaller / larger?

Put your cursor on top of the skin you want to resize and scroll.

How do I make the player detect music from the browser?

Install the WebNowPlaying browser extension

How do I make the player detect Spotify?

Enable the Show desktop overlay when using media keys setting in Spotify settings.

How do I deactivate _____, I only want the clock?

Read the Using Rainmeter guide. Mondtholomewtholomew is made out of multiple skins, each of which can be unloaded or loaded independently.

How do I change _____?

Read the Changing settings guide.

How do I install Mondtholomewtholomew?

Please refer to the Home page of the wiki.

How do I get help with Mondtholomewtholomew?

Please first read this wiki. If the answer is not found or your issue is related to Rainmeter itself, join the Rainmeter discord and ask in the #help channel. I would prefer if you used this method of getting help, instead of opening an issue here on GitHub.

How do I report a bug?

Open an issue here on GitHub and describe the bug and the steps to reproduce it.

How do I make a feature request?

Open an issue here on GitHub and describe the way you would want the feature to function and the reason why it should be added (eg. why it's useful).

How can I contribute to Mondtholomewtholomew?

You may fork Mondtholomewtholomew, develop it and make a Pull Request. I'm willing to pull improvements, fixes and features but if you're adding new stuff, it would be better for both of us if you first made a feature request explaining what you intend to do.
